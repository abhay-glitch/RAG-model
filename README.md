# RAG-model
this is a RAG based model that have using LLM , langchain , open source model
Artificial Intelligence (AI) is the simulation of human intelligence in machines. 
It enables systems to perform tasks that normally require human intelligence.

Machine Learning (ML) is a subset of AI that allows computers to learn from data 
and improve their performance over time without being explicitly programmed.

Deep Learning (DL) is a specialized subset of ML that uses neural networks with 
many layers to model complex patterns in data, especially useful for images, audio, and text.

Natural Language Processing (NLP) is a field of AI that enables machines to 
understand, interpret, and generate human language.

ChromaDB is an open-source vector database that stores embeddings and enables 
semantic search and retrieval of relevant information.

LangChain is a powerful framework that helps connect large language models (LLMs) 
with external knowledge sources like databases, APIs, and documents. 
It simplifies building Retrieval-Augmented Generation (RAG) pipelines.

Retrieval-Augmented Generation (RAG) is an approach that combines retrieval of 
relevant documents with language model generation to produce accurate, 
context-aware responses.

Benefits of RAG:
- Reduces hallucinations from LLMs.
- Ensures up-to-date knowledge without retraining the model.
- Makes LLMs more reliable for specialized domains.

Applications of RAG:
- Chatbots and customer support.
- Document summarization.
- Knowledge assistants.
- Academic and research tools.
